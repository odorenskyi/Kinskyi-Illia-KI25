# Лабораторна робота № 8

Тема Реалізація статичних бібліотек модулів лінійних обчислювальних процесів

Мета Закріплення на практиці принципів модульного програмування та методу функціональної декомпозиції. Отримання практичного досвіду створення статичних бібліотек мовою C/C++, роботи з основними типами даних, арифметичними/логічними операторами та потоковим вводом/виводом. Засвоєння основ модульного (Unit) та системного тестування програмного забезпечення у середовищі розробки Code::Blocks.

Завдання 1.	Створити статичну бібліотеку libModulesKinskyi.a, яка включає функцію для математичних розрахунків згідно з підзадачею 8.1.
2.	Розробити консольну програму Kinskyi_task8.exe для вирішення підзадачі 8.2, яка буде використовувати створену бібліотеку.
3.	Провести перевірку працездатності коду за допомогою методів модульного та системного тестування.
